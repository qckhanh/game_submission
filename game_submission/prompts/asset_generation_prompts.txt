# Asset Generation Prompts

## Visual Design Concepts
**Prompt**: "Tạo color scheme và visual style cho game về sustainable city management. Cần professional nhưng friendly, phù hợp với audience Việt Nam."

**Response**:
- Primary: Blue-purple gradient (trust, innovation)
- Secondary: Green tones (environment, growth)
- Accent: Orange (economy, energy)
- Clean, modern design với rounded corners
- Emoji icons cho accessibility và universal appeal

## Icon and Emoji Selection
**Prompt**: "Chọn emoji phù hợp cho các elements trong game: Economy, Environment, Happiness, và các policy cards."

**Response**:
- Economy: 💰 (universal money symbol)
- Environment: 🌿 (nature, growth)
- Happiness: 😊 (positive emotion)
- Policy icons: 🏢🛍️🏭☀️🚌🌳♻️🏥🎓🏠⚽🤖💡🌺🎰🛢️☢️

## UI/UX Design Language
**Prompt**: "Thiết kế UI components cho card game: policy cards, stat bars, buttons. Cần intuitive và responsive."

**Response**:
- Card design: White background, shadow, hover effects
- Stat bars: Gradient fills với animated transitions
- Buttons: Gradient backgrounds, hover animations
- Color coding: Green (positive), Red (negative)
- Typography: Clean sans-serif fonts

## Responsive Design Considerations
**Prompt**: "Game cần chạy tốt trên mobile và desktop. Thiết kế responsive layout cho cards và UI elements."

**Response**:
- CSS Grid cho card layout (auto-fit, minmax)
- Flexbox cho button groups
- Scalable typography (rem units)
- Touch-friendly button sizes (44px minimum)
- Breakpoints: 768px cho tablet/mobile
