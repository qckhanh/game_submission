# Refinement and Testing Prompts

## Gameplay Balance Refinement
**Prompt**: "Game hiện tại có vẻ quá dễ/khó. Làm sao để adjust card effects và win conditions cho balanced difficulty curve?"

**Response**:
- Increase negative effects của economic cards
- Add more trade-offs cho environmental policies
- Adjust starting stats từ 50 về 45 để tăng challenge
- Win condition: All stats ≥ 50 (không phải 40)
- Thêm high-risk/high-reward cards

## User Experience Improvements
**Prompt**: "Người test feedback rằng không rõ card effects sẽ như thế nào. Cải thiện UI/UX để clearer feedback."

**Response**:
- Thêm action feedback modal sau mỗi card selection
- Color-coded effects (green positive, red negative)
- Preview effects trước khi confirm
- Better visual hierarchy trong card design
- Animated stat changes

## Performance Optimization
**Prompt**: "Game bị lag trên mobile devices. Optimize JavaScript và CSS performance."

**Response**:
- Reduce DOM queries bằng caching elements
- Use CSS transforms instead of changing layout properties
- Debounce rapid interactions
- Optimize card rendering với document fragments
- Minimize reflows với batch updates

## Accessibility Enhancements
**Prompt**: "Làm game accessible cho users với disabilities. Cần keyboard navigation và screen reader support."

**Response**:
- Add ARIA labels cho interactive elements
- Keyboard navigation với Tab/Enter
- High contrast mode option
- Text alternatives cho emoji icons
- Focus indicators rõ ràng

## Bug Fixes and Edge Cases
**Prompt**: "Tìm và fix các bugs tiềm ẩn: negative stats, rapid clicking, game state corruption."

**Response**:
- Clamp stats between 0-100
- Disable buttons during transitions
- Validate game state after each action
- Add error boundaries và graceful fallbacks
- Test edge cases: all stats = 0, rapid year progression

## Localization Considerations
**Prompt**: "Chuẩn bị code structure để dễ dàng translate sang English hoặc ngôn ngữ khác."

**Response**:
- Extract all text strings thành constants object
- Separate content từ logic
- Unicode support cho special characters
- RTL language consideration trong CSS
- Flexible text lengths trong UI design
