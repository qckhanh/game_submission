# Code Generation Prompts

## HTML Structure
**Prompt**: "Tạo HTML structure cho game có 4 screens: Menu, Instructions, Game, Results. Cần semantic HTML và accessibility features."

**Response**:
- Screen-based architecture với CSS classes
- Semantic elements: header, main, section
- ARIA labels cho screen readers
- Vietnamese text content
- Clean, nested structure

## CSS Styling System
**Prompt**: "Viết CSS cho modern game interface. Cần animations, responsive design, gradient backgrounds, card hover effects."

**Response**:
- CSS Custom Properties cho theme management
- Flexbox/Grid layouts
- CSS transforms cho animations
- Smooth transitions (0.3s ease)
- Mobile-first responsive design
- Hardware-accelerated animations

## JavaScript Game Logic
**Prompt**: "Implement game state management, card system, và UI updates cho turn-based card game. Cần clean architecture và maintainable code."

**Response**:
- Centralized gameState object
- Event-driven architecture
- Functional programming approach
- Modular functions với single responsibility
- DOM manipulation best practices
- Error handling và validation

## Game Balance Implementation
**Prompt**: "Code hệ thống policy cards với different effects. Cần random card drawing, effect calculation, win/lose conditions."

**Response**:
- Card database với structured effects
- Fisher-Yates shuffle algorithm
- Mathematical stat calculations
- Conditional logic cho game states
- UI feedback systems

## Mobile Optimization
**Prompt**: "Tối ưu JavaScript performance cho mobile browsers. Cần efficient DOM updates và smooth animations."

**Response**:
- Debounced event handlers
- Batch DOM updates
- CSS-based animations over JS
- Minimal reflows/repaints
- Touch event optimization
